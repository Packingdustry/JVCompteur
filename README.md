# JVCompteur
Script Tampermonkey qui numérote les messages d'un topic JVC

## Installation
Vous avez besoin de Tampermonkey : 
* [Mozilla Firefox et dérivés](https://addons.mozilla.org/fr/firefox/addon/tampermonkey/)
* [Google Chrome et dérivés](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)

Puis cliquez sur [ce lien](../../raw/main/dist/JVCompteur.user.js).

## ⚠️ Avis ⚠️
Script développé à l'aide de l'IA générative Gemini 3.1 Pro, peut contenir des erreurs.

Le script est aussi susceptible de casser suite à des mises à jours du site [JeuxVideo.com](https://www.jeuxvideo.com/forums/0-51-0-1-0-1-0-blabla-18-25-ans.htm).

## Contributions
Toute contribution est bienvenue, faites une Pull Request et décrivez ce qu'ajoute votre mise à jour et celle-ci sera verifiée puis potentiellement merge au script.
